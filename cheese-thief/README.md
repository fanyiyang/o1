# 🧀 奶酪大盗 Cheese Thief（在线版 · 4–8 人）

4–8 人社交推理小游戏：找出偷走奶酪的**大盗**。按官方《奶酪大盗》规则实现。纯静态网页 + WebRTC 玩家直连（PeerJS），**无后端、无账号**，托管在 GitHub Pages 上。

🎮 **在线试玩**：https://fanyiyang.github.io/cheese-thief/

## 怎么玩

1. 一名玩家点 **创建房间**，拿到房间号（如 `CHS-7K2`）；可在大厅切换 **偷看规则（默认开）**。
2. 把房间号（或「复制房间号链接」）发给朋友，其他人填昵称 + 房间号 **加入房间**。
3. 满 **4–8 人**房主点 **开始游戏**：每人 1 张隐藏身份（🐭 睡鼠 / 🧀 奶酪大盗）和骰子（开偷看 2 颗 / 关偷看 1 颗）。
4. **选醒来之夜**：点数不同的睡鼠挑一个点数睁眼；大盗两点不同则两晚都睁、相同睁一次。
5. **夜晚**（数 1→6，每晚 10 秒 + 声音）：
   - 轮到你点数的那一晚你会睁眼。**同一晚睁眼的人能互相看到对方睁眼**。
   - 大盗在自己选的那晚拿走奶酪——**同晚睁眼的人会看到是他拿的**（关键线索）。
   - 偷看开 + 你**独自**睁眼时，可点桌上一个头像偷看他的一颗点数。
6. **共犯（5–8 人）**：5 人=和大盗同晚睁眼者成为共犯；6/7/8 人=数完第6晚后大盗再睁眼挑 1/2/2 名共犯。共犯与大盗共享胜利。
7. **白天**：可用**内置语音**（右下角小麦克风，夜晚自动静音）或自己的微信/Discord 讨论。
8. **投票**：每人同时指认一人。**票最多者出局；平票则全部出局**，翻身份牌。
   - 投出大盗 → 🐭 睡鼠胜利；投出睡鼠**或共犯** → 🧀 大盗阵营胜利。

> 右下角有 **📜 我的记录**（回看你这局的经历）和 **🎙️ 语音**（可选）。
> 规则来源：官方《奶酪大盗 Cheese Thief》规则书（含 4 人变化 + 5–8 人共犯）。

## 技术说明

- 纯 `HTML/CSS/JS` 单页，无构建步骤。
- 联网用 [PeerJS](https://peerjs.com/) 玩家直连：**房主的浏览器当“裁判”**，只把每个人自己的身份/骰子私下发给本人，别人的身份不会传到你电脑上。
- 信令走 PeerJS 免费公共服务器；数据点对点传输。

### 本地开发

```bash
# 跑纯逻辑单元测试
npm test

# 本地预览（任选其一）
python3 -m http.server 8091
# 然后浏览器打开 http://localhost:8091
```

### 已知限制

- 房主掉线则本局作废回大厅；中途掉线不支持重连（重开一局即可）。
- 内置语音是点对点：极少数对称 NAT 网络可能连不上，这种就退回用微信/Discord。
- 游戏进行中不接受新玩家加入（等本局结束）。
